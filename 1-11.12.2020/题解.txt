4.迷宫
DFS大法好，有一个更方便的DFS方法：
int dfs(int x,int y)
{
	if(mmp[x][y])return 0;					//顺序很重要 
	else if(x==ed_x && y==ed_y)return 1;
	mmp[x][y]++;
	int tp=dfs(x+1,y)+dfs(x-1,y)+dfs(x,y+1)+dfs(x,y-1);
	mmp[x][y]--;
	return tp;
}
坑爹的是，终点可能有障碍，卡了我10分。
