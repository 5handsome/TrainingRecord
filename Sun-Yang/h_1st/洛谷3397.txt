题目描述
在n×n 的格子上有m 个地毯。
给出这些地毯的信息，问每个点被多少个地毯覆盖。
输入格式
第一行，两个正整数n,m。意义如题所述。接下来m 行，每行两个坐标 代表一块地毯的左上角右下角

输出格式
输出n 行，每行 n 个正整数。
第i 行第 j 列的正整数表示 (i,j) 这个格子被多少个地毯覆盖。

题目分析
	存图 两个循环让数组里的加1就行



#include<iostream> 
using namespace std;	


int a[1001][1001] = {0};

int main()
{
	int n,nn;int sx,sy,x,y;int xx,yy;
	cin >> n >> nn;
	for(int i = 0; i < nn; i ++)
	{

		cin >> sx >> sy >> x >> y;
		for(xx = sx;xx < x+1;xx ++)
			for(yy = sy; yy < y+1;yy ++)
				a[xx][yy] += 1;
		
	}
	
	for(int j = 1;j < n+1;j ++)
	{
		for(int kk = 1;kk < n+1;kk ++)
		{
			cout << a[j][kk] << " ";
		}
		cout <<'\n';
		
		
	}
}
